# Getting Started with Create MoveMyBike

## Available Scripts

In the project directory, you can run:

### `npm start`
  #### `local`
 ##### `npm run start:local`
  #### dev - npm run start:dev
   #### prod - npm run start:prod
   
### `npm run build` fails to minify
#### dev - npm run build:dev
#### prod - npm run build:prod
